# NJY 做题微信小程序前端

这是“南检院做题”微信小程序前端 MVP，基于原生微信小程序实现。

## 功能范围

- 首页教学视频和学习推广内容
- 授权题库列表、章节列表、离线缓存入口
- 练习设置、做题、题目导航、答案解析
- 套卷列表、考试说明、考试结果
- 错题本、题目收藏、趋势统计、考试成绩
- 我的页面、登录入口、奖牌、排名、工具区
- 反馈、设置、协议页面

## 后端接口

统一后端地址：

```text
https://www.njwjxy.cn:30443/api/miniapp
```

当前页面已通过 `utils/services.js` 对接真实后端接口。后端不可用时，部分展示型页面会使用本地模拟数据兜底；登录失败、未授权等权限错误不会被模拟数据覆盖。

## 安全说明

- 小程序 AppID 写在 `project.config.json` 和 `app.js` 中。
- AppSecret 不允许写入前端仓库。
- 所有密码、token、数据库连接串和对象存储密钥都不得明文提交。
